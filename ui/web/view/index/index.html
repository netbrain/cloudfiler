Welcome to CloudFiler

CloudFiler is a a simple web based access controlled file archive. Upload files and share them with users on a per user or group level.

To get started, upload a file and/or view your available files

Features

  • Administration of users and their access to files.
  • User level and/or role level based access.
  • Multiple file upload solution.
  • Add metadata to files through file description or file tags.
  • File search functionality that leverages free text searching on filename and description and tags.
  • Rich user-interface: filter and sort files in realtime on client side through javascript.
  • Built using Google Go, making it extremely fast and scalable!
